# Glossário

- **Asset pack/expansão:** pacote cosmético de imagens, áudio, fundos e ícones.
- **Challenge key:** identificador canônico do período, por exemplo `daily:2026-06-27`.
- **Core/simulation:** regras puras, independentes de Phaser, DOM, rede e storage.
- **Entity definition:** dado estável que define regras de uma entidade.
- **Hitbox:** geometria competitiva; não é derivada automaticamente dos pixels do sprite.
- **Run:** uma partida desde o spawn até game over.
- **Ruleset version:** versão das regras que pode afetar replay/placar.
- **Seed:** entrada determinística do PRNG.
- **Tick:** passo fixo de simulação, 1/60 s.
- **Visual bounds:** área visual do asset; pode diferir da hitbox.
